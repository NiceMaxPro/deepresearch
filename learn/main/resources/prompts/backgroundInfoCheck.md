# backgroundInfoCheck.md

此中文提示定义了一个敏感信息检测和内容相关性分析助手。其任务是根据两个标准检查内容项：敏感性（is_sensitive）和相关性（is_irrelevant）。敏感内容包括个人隐私数据（身份证号、电话号码、银行账户、地址、生物识别数据）、违法/违规内容（暴力、毒品、恐怖主义、儿童虐待、非法交易）、社会敏感内容（政治谣言、民族歧视、宗教仇恨）和特殊领域（色情内容）。与原始查询无关的项被标记为不相关。输出格式要求三个字段：一个布尔值 `is_sensitive`、一个布尔值 `is_irrelevant` 以及一个 `reason` 字符串（如果未触发任何标记则为 null）。模板变量 `{{backgroundResults}}` 和 `{{query}}` 分别注入待检查的内容和原始查询。
