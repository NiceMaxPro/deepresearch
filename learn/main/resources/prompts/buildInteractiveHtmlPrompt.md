# buildInteractiveHtmlPrompt.md

此中文提示模板定义了一个交互式报告生成专家角色，负责将深度研究报告转换为具有动态可视化效果的响应式单页应用（SPA）HTML 页面。它指定了一种非线性的、基于主题的 SPA 架构，包括固定的顶部导航栏、展示有冲击力数据图表的首屏页面、主题化模块卡片布局（至少 4 个核心主题）以及数据来源页脚。定义了一个三级交互系统：通过带悬停颜色动画的主题色块导航实现视觉引导，通过可展开卡片和标签切换实现内容探索，以及通过带悬停详情和筛选控件的交互式图表实现数据探索。

提示指定了严格的技术栈：Tailwind CSS（CDN）、Chart.js（CDN）、Google Fonts（Noto Sans SC）、移动优先的响应式标准（375px 到 1024px），并禁止使用 SVG 矢量图形、Mermaid 流程图和外部图片。内容映射规范展示了如何从研究报告中提取核心模块并将其映射到交互式 HTML 结构。质量检查点要求至少 5 个主题模块、三级导航系统、移动端折叠菜单、主题色块导航、卡片悬停效果（Y 轴位移配合阴影扩展）以及带注释的数据图表。最终输出必须是一个完整的、独立的 HTML 文件，不包含 Markdown 包装器，不包含解释性文字，并且必须通过 W3C 验证，FCP 在 1.2 秒以内。
