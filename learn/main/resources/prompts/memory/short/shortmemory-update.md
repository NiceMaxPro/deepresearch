# shortmemory-update.md

此提示模板定义了一个短期记忆更新代理，负责管理对话过程中用户角色记忆的演变。它比较三个记忆对象：当前提取记忆（latest extraction）、上一次提取记忆（last stored memory）和历史提取追踪记录（chronological record of all past extractions）。代理执行两种操作之一：UPDATE（当前记忆和上一次记忆非常相似时合并特征，提高置信度级别）或 NO CHANGE（当前记忆与上一次记忆以及历史追踪记录中的大多数记录存在显著偏差时）。

更新决策使用来自 `{{ update_similarity_threshold }}` 模板变量的相似度阈值。合并时，代理对相似术语保留更具体的描述（如"高中生"优于"学生"），对有一定差异的术语则两者都保留。仅当当前提取和上一次提取存在显著差异时，才会参考历史追踪记录，为决策提供更广泛的上下文。输出遵循与提取代理相同的 JSON 结构，包含合并后的证据摘要和调整后的置信度级别。提示中包含了详细的示例，展示了 UPDATE 和 NO CHANGE 两种场景，包括用户从技术问题转向育儿问题的案例。
