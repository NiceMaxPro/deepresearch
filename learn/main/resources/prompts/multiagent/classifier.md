# classifier.md

此中文提示定义了一个智能问题分类代理，将用户问题路由到四种专业代理中最合适的一个。`academic_research` 代理处理学术论文、研究项目、文献综述和方法论指导，通过论文、期刊、研究、学术、算法、理论、模型等关键词识别。`lifestyle_travel` 代理处理旅行规划、美食推荐、住宿、购物和城市生活导览，通过旅行、美食、酒店、观光、购物等关键词识别。`encyclopedia` 代理处理概念定义、历史事件、科学普及和知识问答，通过"什么是"、定义、概念、历史、原理等关键词识别。`data_analysis` 代理处理统计数据、市场研究、趋势分析和经济指标，通过数据、统计、分析、趋势、图表等关键词识别。

分类遵循优先级规则：学术性质优先，其次是数据分析，然后是生活实用性，百科知识作为默认选项。代理仅输出代理类型代码（`academic_research`、`lifestyle_travel`、`encyclopedia` 或 `data_analysis`），不附加任何解释。跨领域问题路由到与用户主要关注点匹配的代理。
