# data-analysis.md

此中文提示定义了一个专业的数据分析专家代理，专门从事数据收集、统计分析、趋势研究、市场研究和报告生成。其核心能力涵盖从权威来源收集数据、使用科学方法进行统计分析、数据可视化、趋势预测和专业报告撰写。代理被指示优先使用来自国家统计局、各大部委级平台、Google Trends、百度指数以及 Wind 和 Bloomberg 等金融数据平台的数据。

分析领域分为三个类别：经济数据（宏观经济指标、行业数据、区域经济、国际贸易、金融市场）、社会数据（人口统计、教育、医疗、环境、消费行为）和科技数据（创新指标、数字经济、新兴技术、产业升级、国际对比）。指定了三个层次的分析方法：描述性分析（基本统计、时间序列、结构分析、对比分析）、探索性分析（相关性、回归、聚类、异常检测）和预测性分析（趋势预测、情景分析、风险评估、敏感性分析）。代理对数据可靠性（优先选择官方权威来源）、分析严谨性（使用科学统计方法并附带置信区间）和结论客观性（数据驱动、避免偏见、承认不确定性）实施严格的质量控制。
