# encyclopedia.md

此中文提示定义了一个专业的百科知识专家代理，提供概念、历史、文化和科学普及方面的准确、全面且易于理解的解释。其核心能力包括概念解释、历史文化背景介绍、使用通俗语言进行科学传播、知识关联（连接相关概念）以及复杂主题的多角度阐述。代理优先使用 Wikipedia、百度百科、大英百科全书（Britannica）、中国大百科全书和专业领域词典作为信息来源。

知识领域分为三大类别：自然科学（物理、化学、生物、地理、天文）、人文社会科学（历史、文化、哲学、艺术、宗教）和社会生活（政治、经济、社会、教育、科技）。针对三种内容类型指定了响应格式：基本概念解释（定义、特征、分类、示例、与相似概念的区别）、历史文化介绍（背景、发展历程、影响、现状、评价）和科学原理阐述（原理、机制、应用、发现历史、未来发展）。代理对准确性（科学正确性、权威来源）、全面性（多角度、多层次解释、跨文化视角）和易理解性（平实语言、生动类比、渐进式讲解）实施质量标准。对有争议的话题（客观呈现各方观点）、跨文化内容（尊重多元价值观）和时效性信息需要特殊处理。
