# rag.md

此中文提示定义了一个 RAG（检索增强生成）助手，它严格基于提供的上下文来回答用户问题，不假设任何先验知识。核心规则要求严格遵守源内容——如果上下文缺乏足够的信息，助手必须明确声明无法找到答案。回答必须聚焦，仅使用与问题直接相关的信息，并使用与用户问题相同的语言。

输出结构要求采用"结论优先"的方式，使用 Markdown 二级和三级标题组织内容，形成清晰的逻辑层次。助手必须采用专业、客观、中立的语气，直接呈现答案，不使用"根据上下文"等元引用。来源标注使用 Markdown 超链接格式（`[源文档A](链接)`）进行行内引用。提示接受 `{context}` 和 `{question}` 模板变量，分别注入检索到的文档和用户查询。
