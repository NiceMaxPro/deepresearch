# researcher.md

此提示模板定义了 `researcher` 代理，由 `supervisor` 代理管理，专门使用内置和动态加载的工具进行彻底的调查。它描述了两种类型的可用工具：内置工具（`searchFilterTool` 用于网页搜索，返回标题、链接、内容和范围在 [-1.0, 1.0] 的信任权重系数，以及 `jinaCrawler` 用于以 Markdown 格式读取 URL 内容）和动态加载的工具（专业化搜索工具、谷歌地图工具、数据库检索工具）。信任权重系数指南指示代理将高信任度的结果放置在消息列表的两端，而将低权重项放在中间。

工作流程包括理解问题、评估可用工具、规划解决方案、使用工具执行（优先选择专业化工具而非通用工具，在有益时组合多个工具）以及综合信息。输出格式与 background.md 的研究者提示一致：问题陈述（Problem Statement）、研究发现（Research Findings，按主题组织，无行内引用）、结论（Conclusion）和作为 Markdown 超链接的参考文献（References）。提示强制执行相同的约束：禁止数学运算、禁止文件操作、仅在不必要的信息缺失时才进行爬取、所有信息必须注明来源、图片仅来自搜索或爬取的内容、并以 `{{ locale }}` 变量指定的语言输出。
